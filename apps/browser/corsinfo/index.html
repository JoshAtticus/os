CORS info

CORS (Cross Origin Resource Sharing) is a security measure commonly used to protect websites against malicious access. However, it can also have unintended side effects, for example, preventing this browser from loading commonly accessed websites, such as Google.

In order to bypass this, we need to use a browser extension which allows us to bypass it. This only works on desktop!!!

Firefox Extension

If you're on Firefox, go ahead and download the CORS Everywhere extension, pin it to your toolbar, click it so it turns green, then refresh this page.

Chrome Extension

If you're on Chrome (or a Chromium-based browser, such as Vivaldi or Microsoft Edge), go ahead and download the Allow CORS extension, pin it to your toolbar, open it, click the grey icon so it shows colour, then refresh this page.
